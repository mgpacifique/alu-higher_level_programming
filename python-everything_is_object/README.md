# Python - Everything is object

This directory contains exercises about Python's object model, identity, mutability, and references.

## Concepts Covered

- Object identity and equality (`is` vs `==`)
- Mutable vs immutable objects
- Object references and aliasing
- Integer caching and string interning
- List and tuple behavior
- Memory addresses and object IDs

## Files

- `0-answer.txt` to `28-answer.txt` - Answers to questions about Python objects
- `19-copy_list.py` - Function to copy a list
