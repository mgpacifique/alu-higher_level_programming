# Python - More Classes and Objects

This directory contains tasks related to advanced Object-Oriented Programming (OOP) concepts in Python.

## Files

- `0-rectangle.py` - Empty class Rectangle that defines a rectangle
- `1-rectangle.py` - Rectangle class with private width and height attributes
- `2-rectangle.py` - Rectangle class with area and perimeter methods
- `3-rectangle.py` - Rectangle class with string representation using #
- `4-rectangle.py` - Rectangle class with __repr__ method for eval()
- `5-rectangle.py` - Rectangle class with __del__ method and deletion message
- `6-rectangle.py` - Rectangle class with instance counter (number_of_instances)
- `7-rectangle.py` - Rectangle class with configurable print_symbol
- `8-rectangle.py` - Rectangle class with static method bigger_or_equal()
- `9-rectangle.py` - Rectangle class with class method square()
