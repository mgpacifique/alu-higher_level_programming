# Python - Inheritance

This directory contains exercises about inheritance in Python, including class inheritance, method overriding, and type checking.

## Concepts Covered

- Object introspection with `dir()`
- Class inheritance
- Method overriding
- Type checking: `type()`, `isinstance()`
- Public and private attributes
- `super()` function
- Abstract base classes

## Files

- `0-lookup.py` - Function that returns list of object attributes and methods
- `1-my_list.py` - MyList class that inherits from list
- `2-is_same_class.py` - Function to check exact class match
- `3-is_kind_of_class.py` - Function to check instance or inheritance
- `4-inherits_from.py` - Function to check inheritance (not exact match)
- `5-base_geometry.py` - Empty BaseGeometry class
- `6-base_geometry.py` - BaseGeometry with area() method
- `7-base_geometry.py` - BaseGeometry with integer_validator() method
- `8-rectangle.py` - Rectangle class inheriting from BaseGeometry
- `9-rectangle.py` - Rectangle with area() and __str__() methods
- `10-square.py` - Square class inheriting from Rectangle
- `11-square.py` - Square with custom __str__() method
- `tests/` - Directory containing test files for doctest
