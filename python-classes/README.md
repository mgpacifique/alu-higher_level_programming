# Python - Classes and Objects

This directory contains tasks related to Object-Oriented Programming (OOP) in Python.

## Files

- `0-square.py` - Empty class Square that defines a square
- `1-square.py` - Square class with private instance attribute size
- `2-square.py` - Square class with size validation
- `3-square.py` - Square class with area method
- `4-square.py` - Square class with getter and setter for size
- `5-square.py` - Square class with my_print method
- `6-square.py` - Square class with position attribute
