===========================
Test cases for MyList class
===========================

This module defines a class MyList that inherits from list.

Usage
=====

Importing the class:
    >>> MyList = __import__('1-my_list').MyList

Creating an instance:
    >>> my_list = MyList()
    >>> type(my_list)
    <class '1-my_list.MyList'>

Testing inheritance:
    >>> isinstance(my_list, list)
    True

Testing with empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Testing with positive integers:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Testing with negative integers:
    >>> my_list = MyList()
    >>> my_list.append(-1)
    >>> my_list.append(-4)
    >>> my_list.append(-2)
    >>> my_list.print_sorted()
    [-4, -2, -1]

Testing with mixed positive and negative integers:
    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.append(-2)
    >>> my_list.append(0)
    >>> my_list.append(-8)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [-8, -2, 0, 3, 5]

Testing with duplicates:
    >>> my_list = MyList()
    >>> my_list.append(3)
    >>> my_list.append(3)
    >>> my_list.append(1)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [1, 3, 3, 3]

Testing with one element:
    >>> my_list = MyList()
    >>> my_list.append(42)
    >>> my_list.print_sorted()
    [42]

Testing with reverse sorted list:
    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.append(4)
    >>> my_list.append(3)
    >>> my_list.append(2)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Testing with already sorted list:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(4)
    >>> my_list.append(5)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
