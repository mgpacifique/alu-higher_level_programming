# Python - Input/Output

This directory contains exercises about file I/O operations and JSON serialization/deserialization in Python.

## Concepts Covered

- Reading and writing files with `open()` and `with` statement
- UTF-8 encoding
- JSON serialization and deserialization
- `json.dumps()` and `json.loads()`
- `json.dump()` and `json.load()`
- Object serialization
- Class dictionary (`__dict__`)
- Pascal's Triangle algorithm

## Files

- `0-read_file.py` - Function that reads a text file and prints it
- `1-write_file.py` - Function that writes a string to a text file
- `2-append_write.py` - Function that appends a string to a text file
- `3-to_json_string.py` - Function that returns JSON representation of an object
- `4-from_json_string.py` - Function that returns object from JSON string
- `5-save_to_json_file.py` - Function that writes object to file using JSON
- `6-load_from_json_file.py` - Function that creates object from JSON file
- `7-add_item.py` - Script that adds arguments to a list and saves to JSON file
- `8-class_to_json.py` - Function that returns dictionary for JSON serialization
- `9-student.py` - Student class with to_json() method
- `10-student.py` - Student class with filtered to_json() method
- `11-student.py` - Student class with reload_from_json() method
- `12-pascal_triangle.py` - Function that returns Pascal's triangle
